Đây là source code của project giữa kỳ môn An toàn - Bảo mật hệ thống thông tin do Lê Võ Khánh Hưng thực hiện, 
dùng để thực hiện mã hóa các file .txt và các file với extension khác (đang thực hiện).

Để chạy chương trình từ Git này, vui lòng chọn Folder Main Components, chạy file MyFrame.java.
Folder Encrypt Method chứa các phương thức để mã hóa file.
Folder Encrypt Panel và folder Decrypt panel chứa các màn hình để thực hiện mã hóa và giải mã file .txt.
